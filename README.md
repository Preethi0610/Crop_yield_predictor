A machine learning-based application designed to forecast crop yields using environmental and agricultural parameters such as weather conditions, soil data, and regional attributes. This tool empowers farmers and agricultural planners to make informed decisions, enhance resource allocation, and optimize crop production.

---

## Features

- Predicts crop yields based on weather patterns, soil types, and other input features.
- Utilizes **Random Forest** algorithm for high accuracy and robustness.
- Hyperparameter tuning using **GridSearchCV**.
- Model evaluation using **Confusion Matrix**, **Accuracy**, and **Classification Report**.
- Clean and intuitive **user interface** for seamless interaction and visualization of predictions.
- Built with **Python** and **scikit-learn**.
